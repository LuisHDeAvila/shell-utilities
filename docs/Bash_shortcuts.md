# shortcuts de bash
```
C	Control
M	Tecla Meta (Alt o ESC)

C-c	Interrumpir el proceso en ejecución
C-z	Suspender el proceso en ejecución
C-d	Salir del shell actual
C-l	Pantalla clara
C-a	Mover al principio de la línea
C-e	Mover al final de la línea
M-←	Retroceder una palabra
M-b	Retroceder una palabra (alternativa)
M-→	Avanzar una palabra
M-f	Avanzar una palabra (alternativa)
C-x, C-x	Mover el cursor entre el inicio de la línea y la posición actual
M-t	Intercambiar las dos últimas palabras antes del cursor
C-t	Intercambiar los dos últimos caracteres antes del cursor
M-Backspace	Eliminar una palabra (hacia atrás)
C-w	Eliminar una palabra (hacia atrás) (alternativa)
M-d	Eliminar una palabra (reenviar)
C-h	Eliminar un carácter hacia atrás (igual que Retroceso)
C-k	Matar desde el cursor hasta el final de la línea
C-u	Matar desde el cursor hasta el comienzo de la línea
M-u	Mayúsculas desde el cursor hasta el final de la palabra
M-l	Minúsculas desde el cursor hasta el final de la palabra
M-c	Poner en mayúscula la palabra (desde la posición del cursor)
C-y	Pegar
C-_	Deshacer
M-r	Volver al comando original (antes de editarlo)
C-p	Obtener comando anterior
C-n	Obtener el siguiente comando
C-r, <pattern>	Buscar comando anterior que contenga <patrón>
M-.	Insertar último argumento del último comando
M-<N>, C-M-y	Obtenga el argumento N del comando anterior (N = 0 da el comando)
M-<	Insertar el comando más antiguo de la historia
C-x,C-v	Mostrar versión bash
```
